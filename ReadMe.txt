
A program using Python that performs the following:

Take any random article on Wikipedia (e.g. http://en.wikipedia.org/wiki/Art)

and click on the first link on the main body of the article that is not within

(parenthesis) or italicized; If you repeat this process for each subsequent

article you will often end up on the Philosophy page.



Solution:

Step to run the program:
1. Install BeautifulSoup: pip install beautifulsoup4

2. Install Requests: pip install requests

3. Run the Program: python wiki.py --> for a single link
		    python wikiFor500links --> for random 500 links
